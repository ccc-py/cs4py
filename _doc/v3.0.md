# cs4py v3.0 開發計畫

> **定位**：在 v1.0（四大領域）+ v2.0（密碼學/資料結構）的基礎上，新增 **數值方法**、**資料壓縮**、**計算幾何**、**資訊檢索**、**機率演算法** 五大領域，並擴充 AI（NLP/推薦系統）與演算法（圖論進階/近似演算法/進階資料結構）。聚焦於「可純 Python 實作的經典演算法」，不碰系統層級。

---

## 一、全新領域（六個頂層目錄）

### 1. `code/數值方法/` — 科學計算的基石

數值方法是工程、物理、金融模擬的基礎，從 NASA 登月軌道計算到現代天氣預報皆依賴之。

| 子目錄 | 檔案 | 演算法 | 歷史意義 |
|--------|------|--------|---------|
| `root_finding/` | bisection.py | 二分法 | 最簡單可靠的數值求根法 |
| `root_finding/` | newton.py | 牛頓法 | 17 世紀牛頓發明，二次收斂 |
| `root_finding/` | secant.py | 割線法 | 無需導數的牛頓法變體 |
| `linear/` | gaussian_elimination.py | 高斯消去法 | 線性代數核心演算法 |
| `linear/` | lu_decomposition.py | LU 分解 | 高效求解多組線性方程 |
| `linear/` | jacobi_iteration.py | 雅可比迭代 | 大型稀疏矩陣求解 |
| `integration/` | trapezoidal.py | 梯形法則 | 數值積分基礎 |
| `integration/` | simpson.py | 辛普森法則 | 更高精度的數值積分 |
| `integration/` | monte_carlo.py | 蒙地卡羅積分 | 高維積分的標準方法 |
| `interpolation/` | lagrange.py | 拉格朗日插值 | 多項式插值經典 |
| `interpolation/` | spline.py | 三次樣條插值 | 平滑曲線擬合標準 |
| `ode/` | euler.py | 尤拉方法 | 常微分方程最簡單解法 |
| `ode/` | runge_kutta.py | 四階龍格-庫塔法 | 工程最常用的 ODE 求解器 |

壓縮演算法是 ZIP、PNG、GIF、JPEG 等格式的核心，直接源於 Shannon 資訊理論。

| 子目錄 | 檔案 | 演算法 | 歷史意義 |

| `entropy/` | shannon_fano.py | Shannon-Fano 編碼 | 1948 年，最早的熵編碼 |
| `entropy/` | arithmetic_coding.py | 算術編碼 | 突破整數位元限制 |
| `dictionary/` | lz77.py | LZ77 | 1977 年，ZIP/GZIP 基礎 |
| `dictionary/` | lzw.py | LZW | 1984 年，GIF/Unix compress |
| `dictionary/` | lz78.py | LZ78 | LZW 的前身 |
| `transform/` | run_length.py | 行程編碼 (RLE) | 最簡單壓縮，BMP/PDF 使用 |
| `transform/` | burrows_wheeler.py | Burrows-Wheeler 變換 | bzip2 核心 |
| `transform/` | move_to_front.py | Move-to-Front 編碼 | 配合 BWT 使用 |
| `image/` | dct.py | 離散餘弦變換 | JPEG 核心 |

### 3. `code/計算幾何/` — 空間演算法的基礎

計算幾何是電腦圖學、CAD、機器人導航、GIS 系統的基礎。1970 年代 Shamos 與 Hoey 奠定此領域。

| 子目錄 | 檔案 | 演算法 | 歷史意義 |

| `basic/` | point.py | 點與向量運算 | 幾何基礎操作 |

| `convex_hull/` | graham_scan.py | 葛拉罕掃描法 | O(n log n) 凸包演算法 |
| `convex_hull/` | jarvis_march.py | Jarvis 行禮包演算法 | 輸出敏感型凸包演算法 |
| `convex_hull/` | quickhull.py | QuickHull | 類快速排序思路 |
| `closest_pair/` | closest_pair.py | 最近點對（分治法） | O(n log n) 經典 |
| `polygon/` | point_in_polygon.py | 點在多邊形內檢測 | 射線法/繞數法 |
| `polygon/` | polygon_area.py | 多邊形面積（鞋帶公式） | 簡單但實用的公式 |
| `polygon/` | triangulation.py | 多邊形三角剖分 | 圖形渲染基礎 |
| `sweep_line/` | line_sweep.py | 掃描線演算法 | 線段相交 counting |
| `triangulation/` | delaunay.py | Delaunay 三角剖分 | 網格生成標準方法 |

| `ranking/` | bm25.py | Okapi BM25 | 現代搜尋引擎 ranking |

### 5. `code/機率演算法/` — 隨機性的力量

機率演算法在 1970 年代興起，解決了確定性演算法難以處理的問題，是大數據、密碼學的核心工具。

| 子目錄 | 檔案 | 演算法 | 歷史意義 |
|--------|------|--------|---------|
| `monte_carlo/` | integration.py | 蒙地卡羅積分 | 高維積分的標準方法 |
| `monte_carlo/` | pi_estimation.py | π 的蒙地卡羅估計 | 經典隨機模擬 |
| `monte_carlo/` | sampling.py | 重要性採樣 | 變異數降低技巧 |
| `randomized/` | quickselect.py | 隨機化快速選擇 | O(n) 期望時間找第 k 大 |
| `randomized/` | randomized_mst.py | 隨機化最小生成樹 | Karger-Klein-Tarjan |
| `randomized/` | skip_list.py | 跳躍列表 | 隨機化平衡搜尋結構 |
| `probabilistic_ds/` | bloom_filter.py | 布隆過濾器 | 空間高效的成員測試 |
| `probabilistic_ds/` | hyperloglog.py | HyperLogLog | 大基數估計 |
| `probabilistic_ds/` | count_min_sketch.py | Count-Min Sketch | 資料流頻率估計 |
| `probabilistic_ds/` | min_hash.py | MinHash | Jaccard 相似度估計 |
| `markov/` | markov_chain.py | 馬可夫鏈模擬 | 隨機過程基礎 |
| `markov/` | mcmc.py | MCMC (Metropolis-Hastings) | 貝氏推論核心方法 |
