# cs4py v1.0 開發紀錄

## 專案概述

cs4py 是一個以 Python 實作資訊科學經典演算法與系統的教學專案，涵蓋計算理論、演算法、人工智慧、程式語言四大領域。每個實作都搭配詳細的歷史背景說明文件，強調教學性與可讀性。

**核心原則：** 純 Python 實作、中文註解、包含測試範例、每個 .py 搭配 .md 說明文件。

## 技術亮點

### 從零實作的核心組件
- **純 Python 神經網路**：CNN（卷積正向/反向傳播）、RNN/LSTM（門控機制）
- **編譯器前端**：Lisp 解譯器（eval/apply）、計算機語言（遞迴下降解析、AST）
- **搜尋與最佳化**：A*、MCTS、模擬退火、遺傳演算法、粒子群優化

### 教學設計
- 每個演算法都從歷史背景出發，解釋「為什麼」而不只是「怎麼做」
- 演示範例涵蓋經典問題（XOR、井字遊戲、N-皇后、手寫數字）
- 所有程式碼可獨立執行（`python3 xxx.py` 即可看到效果）
