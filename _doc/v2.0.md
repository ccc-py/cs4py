# cs4py v2.0 開發計畫

> **定位**：在 v1.0 四大領域（計算理論、演算法、人工智慧、程式語言）的基礎上，新增密碼學與資料結構兩大領域，並擴充既有子領域的經典實作。聚焦於經典演算法與 AI，不涵蓋系統層級實作。

---

## 一、全新領域（兩個頂層目錄）

### 1. `code/密碼學/` — 從古典到現代的加密演算法

密碼學擁有深厚的歷史背景，從古羅馬的凱薩密碼到現代的 RSA 公鑰加密，完美符合專案「歷史角度」的選題原則。

| 子目錄 | 檔案 | 演算法 | 歷史意義 |
|--------|------|--------|---------|
| `classical/` | caesar.py | 凱薩密碼 | 古羅馬時期最早替換密碼 |
| `classical/` | vigenere.py | 維吉尼亞密碼 | 16 世紀多表替換，曾被稱為「不可破解」 |
| `classical/` | frequency.py | 頻率分析攻擊 | 9 世紀 Al-Kindi 發明，破解替換密碼 |
| `classical/` | enigma.py | 恩尼格瑪密碼機 | 二戰密碼戰，圖靈破解 |
| `modern/` | rsa.py | RSA 公鑰加密 | 1977 年 Rivest-Shamir-Adleman，改變世界 |
| `modern/` | diffie_hellman.py | Diffie-Hellman 金鑰交換 | 1976 年，公鑰密碼學開端 |
| `modern/` | elgamal.py | ElGamal 加密 | 1985 年，基於離散對數 |
| `hash/` | sha256.py | SHA-256 | NIST 標準，區塊鏈基礎 |
| `hash/` | md5.py | MD5（簡化版） | 1991 年，廣泛使用但有碰撞問題 |
| `signature/` | digital_signature.py | RSA 數位簽章 | 電子商務與電子簽章基礎 |

### 2. `code/資料結構/` — CS 最核心的基礎

資料結構是演算法的基礎設施，獨立成區可讓學習路徑更清晰。

| 子目錄 | 檔案 | 資料結構 | 核心操作 |
|--------|------|---------|---------|
| `linked_list/` | singly.py | 單向鏈結串列 | insert/delete/reverse/merge |
| `linked_list/` | doubly.py | 雙向鏈結串列 | 前後遍歷、LRU Cache |
| `tree/` | binary_tree.py | 二元樹遍歷 | 前/中/後序、層序 |
| `tree/` | bst.py | 二元搜尋樹 | insert/search/delete |
| `tree/` | avl.py | AVL 樹 | 自我平衡旋轉 |
| `tree/` | red_black.py | 紅黑樹 | 紅黑性質維護 |
| `hash_table/` | chaining.py | 鏈結分離雜湊表 | 碰撞處理 |
| `hash_table/` | open_addressing.py | 開放定址（線性/二次探測） | 載入因子與再雜湊 |
| `heap/` | binary_heap.py | 二元堆積 | push/pop/heapify |
| `union_find/` | uf.py | 並查集 | 路徑壓縮、按秩合併 |

## 二、既有領域擴充

### 演算法/graph/ — 高優先級

| 檔案 | 演算法 | 應用場景 |
|------|--------|---------|
| `topological_sort.py` | 拓撲排序（Kahn's / DFS） | 編譯器依賴分析、任務排程 |
| `strongly_connected.py` | 強連通分量（Kosaraju / Tarjan） | 社交網路分析 |
| `max_flow.py` | 最大流（Ford-Fulkerson / Edmonds-Karp） | 網路流量、匹配問題 |
| `articulation.py` | 關節點與橋（Tarjan） | 網路可靠性分析 |
| `eulerian.py` | 歐拉路徑/迴路 | 七橋問題、DNA 組裝 |
| `bipartite_matching.py` | 二分圖匹配（匈牙利演算法） | 任務分配、穩定婚姻 |

### 演算法/dp/ — 中優先級

| 檔案 | 演算法 | 重要性 |
|------|--------|--------|
| `edit_distance.py` | 編輯距離（Levenshtein） | 拼字檢查、生物資訊 |
| `lis.py` | 最長遞增子序列 | 經典 DP，O(n log n) 優化 |
| `coin_change.py` | 硬幣找零 | 完全背包變體 |
| `subset_sum.py` | 子集和問題 | NP-complete 的 DP 解法 |
| `optimal_bst.py` | 最優二元搜尋樹 | DP 進階應用 |

### 演算法/sort/ — 中優先級

| 檔案 | 演算法 | 備註 |
|------|--------|------|
| `insertion_sort.py` | 插入排序 | 自適應，小陣列高效 |
| `selection_sort.py` | 選擇排序 | 教學基礎 |
| `counting_sort.py` | 計數排序 | 線性時間排序 |
| `bucket_sort.py` | 桶排序 | 分布排序 |
| `shell_sort.py` | 希爾排序 | 插入排序的改進 |

### 演算法/math/ — 中優先級

| 檔案 | 演算法 | 應用 |
|------|--------|------|
| `modular_exponentiation.py` | 模冪運算 | RSA 核心 |
| `miller_rabin.py` | Miller-Rabin 質數測試 | 大質數檢測 |
| `chinese_remainder.py` | 中國剩餘定理 | 古代數論瑰寶 |
| `discrete_log.py` | 離散對數（Baby-step Giant-step） | 密碼學基礎 |

### 人工智慧/ml/ — 中優先級

| 檔案 | 演算法 | 重要性 |
|------|--------|--------|
| `naive_bayes.py` | 樸素貝葉斯 | 機率分類器代表 |
| `linear_regression.py` | 線性迴歸（梯度下降） | 統計學習入門 |
| `logistic_regression.py` | 邏輯斯迴歸 | 二元分類基礎 |
| `pca.py` | 主成分分析 | 降維標準方法 |

### 人工智慧/agent/ — 高優先級

| 檔案 | 演算法 | 重要性 |
|------|--------|--------|
| `q_learning.py` | Q-learning | 無模型強化學習基礎 |
| `sarsa.py` | SARSA | On-policy 強化學習 |
| `bdi_agent.py` | BDI 架構 | 信念-慾望-意圖模型 |
