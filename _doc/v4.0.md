# cs4py v4.0 開發計畫

> **定位**：v3.0 完成了數值/壓縮/幾何/檢索/機率五大領域，v4.0 將加入 v3.0 刻意排除但現在認為**可用 Python 良好模擬**的主題：電腦圖學、區塊鏈、量子計算。同時新增資訊理論、分散式演算法、形式方法，將 cs4py 推向**完整的 CS 知識體系**。
>
> **排除原則**：特別適合 C 語言實作的系統程式主題（編譯器、作業系統模擬）**不納入 v4.0**。這些領域需要底層記憶體控制、指標操作、系統呼叫等，C/Rust 是更合適的選擇。Python 的優勢在演算法演示而非系統實作。
