# 費波那契數列 (Fibonacci Sequence)

## 歷史背景

費波那契數列由義大利數學家李奧納多·費波那契（Leonardo Fibonacci）在 1202 年的著作《計算之書》（Liber Abaci）中引入歐洲數學界。雖然印度數學家更早之前就已經知道這個數列，但費波那契通過「兔子繁殖問題」讓這個數列廣為人知。

### 兔子繁殖問題

假設一對剛出生的兔子（一雄一雌）：
- 兔子在出生後第二個月才能繁殖
- 每對成熟兔子每個月產生一對新兔子
- 兔子永遠不會死亡

問：n 個月後有多少對兔子？

答案就是費波那契數列：1, 1, 2, 3, 5, 8, 13, 21, 34, ...

## 數學定義

```
F(0) = 0
F(1) = 1
F(n) = F(n-1) + F(n-2), for n ≥ 2
```

## 計算方法比較

### 1. 純遞迴

最直觀但效率極低的方法，時間複雜度為 O(2^n)，因為存在大量重複計算。

### 2. 記憶化（自頂向下 DP）

使用字典記錄已計算的結果，避免重複計算。時間複雜度 O(n)，空間複雜度 O(n)。

### 3. 迭代（自底向上 DP）

從小到大計算，只保留前兩個值。時間複雜度 O(n)，空間複雜度 O(1)。

### 4. 矩陣快速冪

利用矩陣乘法：
```
[F(n+1)]   [1  1]^n   [F(1)]
[F(n)  ] = [1  0]   * [F(0)]
```

透過快速冪算法，時間複雜度可達 O(log n)。

### 5. 比內公式（Binet's Formula）

數學閉合解：
```
F(n) = (φ^n - ψ^n) / √5
其中 φ = (1+√5)/2 ≈ 1.618（黃金比例）
     ψ = (1-√5)/2 ≈ -0.618
```

## 數列性質

- **黃金比例**：相鄰兩項的比值 F(n+1)/F(n) 趨近於黃金比例 φ ≈ 1.6180339887...
- **卡西尼恆等式**：F(n-1)×F(n+1) - F(n)² = (-1)^n
- **任意整數表示**：任何正整數都可以表示為不連續的費波那契數之和（齊肯多夫定理）

## 應用場景

1. **計算機科學**：算法分析、資料結構（斐波那契堆）
2. **金融**：斐波那契回調（技術分析工具）
3. **自然科學**：植物的生長模式、貝殼螺旋
4. **藝術與建築**：黃金比例的美學應用
5. **密碼學**：某些加密算法的基礎

## 各方法比較表

| 方法 | 時間複雜度 | 空間複雜度 | 適用場景 |
|------|-----------|-----------|---------|
| 純遞迴 | O(2^n) | O(n) | 僅教學展示 |
| 記憶化 | O(n) | O(n) | 中等 n |
| 迭代 | O(n) | O(1) | 一般用途 |
| 矩陣快速冪 | O(log n) | O(1) | 大 n |
| 比內公式 | O(1)* | O(1) | 理論計算（有浮點誤差） |

*比內公式的實際計算與冪運算和浮點數精度有關
