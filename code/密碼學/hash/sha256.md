# SHA-256 雜湊函數 (SHA-256 Hash Function)

## 歷史背景
SHA-256 由美國國家標準與技術研究院（NIST）於2001年發布，屬於 SHA-2（Secure Hash Algorithm 2）家族。它是 FIPS 180-4 標準的一部分，用於取代 SHA-1。

SHA-256 廣泛應用於：
- SSL/TLS 證書和協議
- 比特幣和區塊鏈技術
- 數位簽章和驗證
- 密碼存儲（配合鹽值）

## 核心原理
SHA-256 將任意長度輸入轉換為固定256位（32字節）的雜湊值：

1. **訊息填充**：附加 bit '1'、填充 '0'、附加長度
2. **初始化**：8個32位常數（質數平方根的小數部分）
3. **壓縮函數**：對每個512位塊進行64輪操作
   - 使用6個邏輯函數：Ch, Maj, Σ₀, Σ₁, σ₀, σ₁
   - 64個常數 K[i]（質數立方根的小數部分）
4. **輸出**：8個32位字串接成256位

## 使用範例
```python
from sha256 import sha256_hex

# 計算雜湊值
message = b"hello world"
hash_value = sha256_hex(message)
print(f"SHA-256: {hash_value}")

# 空字串的雜湊
empty_hash = sha256_hex(b"")
print(f"SHA-256(''): {empty_hash}")
```
